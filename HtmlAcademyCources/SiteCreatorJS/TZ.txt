Техническое задание проекта
Состояние по умолчанию
	Выбран шаблон «Лендинг».
	Блоки шапки, контента и подвала не содержат элементов.
Переключение шаблонов
	В один момент времени может быть выбран только один шаблон.
	Переключение происходит по клику на кнопку с названием шаблона.
	Если пользователь выбрал один шаблон, добавил туда элементы и переключил шаблон, то созданная им версия сайта не сохраняется и не переносится в другой вид шаблона.
Добавление элементов
	При наведении на блок шапки, контента или подвала появляется кнопка-плюсик.
	По клику на эту кнопку на странице появляется панель элементов.
	После клика на кнопке из панели в блоке появляется выбранный элемент.
	Новый элемент добавляется в конец блока.
Редактирование содержимого
	Текстовые элементы
		Каждый новый элемент содержит плейсхолдер — текст по умолчанию.
		Чтобы заменить плейсхолдер на необходимое содержимое, надо кликнуть в контентную область элемента и начать вводить текст.
		Редактирование заканчивается при нажатии клавиши Enter или по клику вне области редактируемого блока.
		Высота элементов растягивается под содержимое.
	Изображения
		По умолчанию блок с изображением содержит иконку с фотоаппаратом.
		По клику на эту иконку появляется возле блока появляется попап с добавлением изображения.
		Ссылка на картинку вставляется в поле ввода в попапе (добавление картинок только по внешним ссылкам).
		Изображение добавляется после нажатия на кнопку «Загрузить».
		Чтобы убрать попап, достаточно кликнуть вне его области.
	Удаление элементов
		По наведению на блок с любым элементом, в левом верхнем углу блока появляется иконка-корзина.
		По клику на эту кнопку блок с элементом удаляется, отменить удаление и вернуть блок нельзя.
